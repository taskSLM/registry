# Contributors

SLM World is built by the community. Every model added, every dataset curated, every task defined, and every line of tooling code makes this registry better for everyone.

## Recognition Tiers

| Tier | Criteria | Perks |
|------|----------|-------|
| **Contributor** | 1+ merged PR | Listed in this file, eternal gratitude |
| **Regular Contributor** | 5+ merged PRs | Listed in the README contributors section |
| **Maintainer** | 10+ merged PRs + community reviews | Merge permissions, issue triage, help shape the project |
| **Emeritus Maintainer** | Former maintainer, inactive 6+ months | Always welcome back, recognized for past service |

## How to Get Listed

1. Make a contribution (model, dataset, task, code, or documentation)
2. Get it merged to `main`
3. Your name is added to this file by a maintainer — or you can add yourself in your PR
